## 🎯 1. Problem & User

### a) เราจะแก้ปัญหาอะไร?
```
แก้ปัญหาการคัดแยกชนิดของมะพร้าวที่ใช้แรงงานคนและเกิด Human Error ได้ง่าย
ซึ่งส่งผลกระทบต่อคุณภาพสินค้า ความเร็วในการผลิต และต้นทุนในห่วงโซ่อุปทาน
```

### b) ใครคือ user หลัก?
```
- เกษตรกรสวนมะพร้าว
- ผู้ประกอบการโรงงานแปรรูป
- แรงงานคัดแยกสินค้า
- ผู้บริโภคที่ต้องการสินค้ามาตรฐาน

ตัวอย่าง:
- โรงงานขนาดเล็กที่ยังคัดแยกด้วยแรงงานคน
- เกษตรกรที่ต้องการลดต้นทุนแรงงาน
- ผู้ประกอบการที่ต้องการเพิ่มความเร็วในการคัดแยก
```

### c) ทำไม Edge AI ถึงเหมาะกับปัญหานี้? (ทำไมไม่ใช่ Cloud)
```
Edge AI สามารถประมวลผลได้บนอุปกรณ์ทันทีโดยไม่ต้องส่งข้อมูลขึ้น Cloud
จึงตอบสนองได้เร็ว (<1 วินาที) ลด latency และใช้งานได้แม้ไม่มีอินเทอร์เน็ต
เหมาะกับการใช้งานในโรงงานหรือพื้นที่เกษตรที่สัญญาณไม่เสถียร
```

---

## 💎 2. Value Proposition

### Unique Value (USP)

```
"เครื่องคัดแยกมะพร้าวด้วย AI ช่วยเกษตรกรและโรงงานคัดแยกมะพร้าวโดยอัตโนมัติ
ต่างจากการคัดแยกด้วยแรงงานคน เพราะทำงานได้รวดเร็ว ลด Human Error
และใช้งานได้แบบ real-time บนอุปกรณ์ Edge AI"
```

## 🛠️ 3. Solution

### a) Hardware ที่จะใช้

- UNO Q + Modulino พื้นฐาน
- เพิ่มเติม:
  - [x] Modulino: Buzzer
  - [x] Servo / Motor: มอเตอร์สายพานลำเลียง
  - [ ] Display: OLED Display
  - [x] อื่นๆ: Webcam / USB Camera

### b) AI Model

- Track: B
- Classes: CO, CM, BG

- จะ improve จาก V2 ตอนนี้อย่างไร?
```
เพิ่ม dataset ในสภาพแสงและมุมกล้องที่หลากหลาย
รวมถึงปรับ preprocessing และ retrain model เพื่อเพิ่มความแม่นยำในการตรวจจับวัตถุ
```

### c) Form Factor

- จะติดที่ไหน? ใช้ยังไง?
```
ติดตั้งกล้องไว้เหนือสายพานลำเลียงมะพร้าว
เมื่อมะพร้าวเคลื่อนผ่านกล้อง ระบบ AI จะตรวจจับและคัดแยกชนิดแบบอัตโนมัติ
จากนั้น buzzer หรือ actuator จะทำงานตาม class ที่ตรวจพบ
เหมาะสำหรับใช้งานในโรงงานขนาดเล็กหรือจุดคัดแยกสินค้าเกษตร
```
